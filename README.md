Welcome to the official **The Little Cavern Studio** repository! 
This collection features custom-engineered tools designed to streamline **REAPER** workflows, focusing on hybrid production, professional mixing, and session management.

---

## 🛠 Included Scripts:

### 🖖 Simple A/B tool for REAPER. Compare your mix with a reference track
This tool is a humble option to ADPTR AB Metric: match levels, and solo frequency bands to check your balance. Requires 4 channels on the Master Track (1/2 Mix, 3/4 Reference).

### 🚀 Dynamic Session Launcher (`TLC_Dashboard.lua`)
The heart of the workflow. A visual dashboard that triggers upon REAPER launch, allowing you to choose session types or open recent projects instantly.
It is very important read About section to learn how configure the startup screen.

### 🏗 Session Builder.lua
The visual configurator for the Launcher. Customize studio names, templates, and button colors without ever touching a single line of code. It automatically manages the `Session_DB.txt` database.

### 🧪 Auto-Assign Analog Molecule.lua
**Matrix Console Companion:** Specifically designed for massive configuration and automatic instance assignment of Analog Molecule. Drastically optimizes routing in hybrid analog/digital environments.

### 🎚 Setlist Manager Pro.lua
Advanced Project Tab management. Ideal for organizing full albums or managing live sets using the asynchronous deployment engine.

### 📐 locate_peaks_item.lua (Math Engine)
High-precision mathematical engine for True Peak localization, ensuring absolute headroom control during mixing and mastering stages.
